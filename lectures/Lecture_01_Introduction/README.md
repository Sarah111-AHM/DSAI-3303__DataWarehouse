## Lecture 1: Introduction to Data Warehousing

**File:** `lectures/Lecture_01_Introduction.pptx`

### Topics Covered:
- What is Data Warehousing? (Centralized repository for BI and reporting)
- Historical Perspective: Bill Inmon (Corporate Information Factory, EDW) vs Ralph Kimball (Dimensional Modeling, Data Marts)
- ETL vs ELT: Extract-Transform-Load vs Extract-Load-Transform
- Staging Area concept: temporary storage for data cleansing before loading
- Data Lake vs Data Warehouse: schema-on-read (raw data) vs schema-on-write (structured)
- Types of Data Warehouses: EDW, ODS, Data Mart
- OLAP vs OLTP: Analytical processing vs Transactional processing
- Database vs Data Warehouse: real-time vs analytical queries
